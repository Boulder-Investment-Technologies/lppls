# lppls
`lppls` is a Python module for fitting the LPPLS model to data.

## Installation
Dependencies

`lppls` requires:
 -pandas >= 0.18.0
 - Python (>= 3.6)
 - NumPy (>= 1.13.3)
 - SciPy (>= 0.19.1)
 - Matplotlib (>= 2.1.1)


User installation
If you already have a working installation of numpy and scipy, the easiest way to install scikit-learn is using pip
```
pip install -U lppls
```
